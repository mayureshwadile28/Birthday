# **App Name**: Papa's Special Day

## Core Features:

- Customizable Greeting: Allow users to compose and edit a personalized birthday message for their father, setting the perfect tone for his special day.
- AI-Powered Sentiment Enrichment Tool: Enhance user-written greetings with AI to suggest better phrasing that is more emotive, and personalized to express gratitude.
- Dynamic Countdown Timer: Implement a real-time countdown displaying the remaining time until September 28, 2025, creating anticipation for the birthday.
- Interactive Photo Gallery: Enable users to upload and display multiple photos in an engaging gallery format, showcasing cherished memories.
- Date Persistence: Allow a user to input the birthday in settings and save this value in localStorage.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to evoke a sense of stability and importance, reminiscent of a deep, comforting night sky.
- Background color: Light cream (#F5F5DC) for a warm, inviting backdrop, soft on the eyes.
- Accent color: Warm gold (#D4AC0D) as an accent to signify the value and richness of the relationship.
- Font pairing: 'Playfair' (serif) for headings and 'PT Sans' (sans-serif) for body text. These create a balance between elegance and readability.
- Use simple, elegant icons that complement the warm color palette.
- A clean and intuitive layout to ensure easy navigation.
- Subtle transitions and animations to enhance user engagement.
